# nn_fund_management

Odoo 17 based fund management project.

## Stack

- **Odoo** 17.0
- **PostgreSQL** 15
- Custom addons under `addons/`

## Getting started

```bash
docker compose up -d
```

Odoo will be available at http://localhost:8069.

## Project structure

```
.
├── addons/                # custom Odoo modules
├── docker-compose.yml     # Odoo + Postgres services
└── README.md
```
